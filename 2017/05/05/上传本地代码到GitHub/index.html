提示：安装 Git 自行解决

1、建立Git仓库

2、添加项目文件到 仓库

3、将 add 的文件commit

4、去github上创建自己的Repository

拿到仓库的 https 地址

例如：https://github.com/ShimingLuo/ShimingLuo.github.io.git

5、将本地仓库关联到 github 上

6、上传之前，先 pull 一下

7、上传到远程仓库

上传没有异常则，上传成功，可能会出现 Username 和 Password，只用输入GitHub的账号和密码就行。
