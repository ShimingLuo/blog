电商平台开发

2018-01-03
  • demo

本次DMEO，仅开发一个具有核心功能的电商平台；
首先选进行准备工作，需求分析、技术选型、环境搭建等；

技术选型

  • 软件过程： 敏捷开发
  • 前后端分离： 完全分离，纯静态方式
  • 模块化方案：CommonJS + Webpack
  • 代理工具： Charles、Fiddler
  • 框架选择： 用户端jQuery + css、管理系统React + Sass
  • 版本控制： git
  • 发布过程： 拉去代码 -> 编译打包 -> 发布到线上机器

环境搭建

  1. git

  2. NodeJs、npm

npm 的初始化、安装/卸载包、自动义命令

  1. Webpack

配置文件处理

  • js 处理 — js-loader

  • css 文件单文件打包引入 — css-loader、style-loader

  • html 模板文件加载 — html-loader

  • 静态资源 — url-loader、file-loader（有些版本不用的吧）

没有设计狮，程序猿。
